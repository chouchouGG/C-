# 未限域enum与限域enum
在C++中所有的`enum`都有一个由编译器决定的**整型的底层类型**。

## 声明差异
**未限域枚举**(unscoped enum)和**限域枚举**(scoped enum)在声明方式上的区别：
```C++
enum Color
{
  black,
  white
}; /* 未限域枚举 */
enum class Color
{
  black,
  white
}; /* 限域枚举 */
```
因此，**限域枚举**又被称为**枚举类**。

## 未限域枚举的两个特点
1. 枚举名会泄露出`enum`作用域。
2. 枚举名会隐式转换为整型。

## 限域枚举
首先，限域枚举消除了未限域枚举的两个特点，其次，其还**支持前置声明**。（未限域枚举也是支持前置声明的不过需要在声明和定义处都注明其底层数据类型）
默认情况下，限域枚举的底层类型是int。
